1. Project Overview

This project focuses on analyzing retail sales data using Tableau to gain insights into sales performance across product sub-categories. The dashboard helps identify trends, top-performing products, and category-wise contributions to total sales.

2. Objectives

Analyze sub-category wise sales performance

Identify top-selling product categories

Visualize sales trends clearly using interactive charts

Support data-driven business decisions

3.Tools Used

Tableau Public / Tableau Desktop

Sample Superstore Dataset

4.Key Features

Stacked bar chart for sub-category wise sales

Interactive filters for better analysis

Clear comparison across product categories

User-friendly and visually appealing dashboard

5.Insights Generated

Identified high-revenue sub-categories

Compared sales distribution across products

Highlighted areas with strong and weak performance

6.Project Deliverables

Tableau workbook (.twb / .twbx)

Interactive sales dashboard

Visual analytics sheets

*Conclusion
This project demonstrates practical skills in Tableau including data visualization, dashboard creation, and sales analysis. It provides meaningful business insights and showcases the ability to convert raw data into actionable information.
